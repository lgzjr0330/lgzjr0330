# 诗意项目管理应用 - 交互设计文档

## 核心交互功能

### 1. 项目管理模块
- **添加项目**: 用户点击"新增项目"按钮，弹出表单包含项目名称、描述、优先级（高/中/低）、分类（工作/学习/生活/创意）、截止日期、标签
- **编辑项目**: 点击项目卡片进入编辑模式，所有字段可修改
- **删除项目**: 项目卡片右上角删除按钮，确认后删除
- **项目筛选**: 左侧边栏提供按分类、优先级、状态的筛选器
- **项目搜索**: 顶部搜索框支持关键词搜索

### 2. 专注计时器模块
- **开始专注**: 选择项目后点击"开始专注"，进入全屏专注模式
- **专注模式**: 显示当前项目名称、计时器、暂停/停止按钮，背景为动态自然景色
- **暂停专注**: 临时暂停计时，可恢复
- **停止专注**: 完全停止专注，可选择保存或放弃本次专注
- **结束专注**: 完成专注，手动输入专注时长或自动记录
- **切换专注**: 快速切换到其他项目继续专注

### 3. 数据统计模块
- **实时统计**: 今日专注时长、本周专注时长、本月专注时长
- **历史记录**: 按日期查看每日专注详情
- **项目统计**: 各项目的累计专注时间
- **分类统计**: 按分类统计专注时间分布
- **时间图表**: 使用ECharts展示专注时间趋势图

### 4. 年度报告模块
- **年度总结**: 每年1月1日生成年度专注报告
- **成就系统**: 根据专注数据解锁诗意成就
- **数据可视化**: 年度专注时间分布、最喜欢的项目、最高效的时段

### 5. 数据管理模块
- **本地存储**: 所有数据存储在浏览器localStorage
- **数据导出**: 支持JSON格式数据下载
- **数据导入**: 支持导入之前导出的数据
- **归墟功能**: 一键清空所有数据和统计，重新开始

### 6. 主题切换模块
- **亮色主题**: 以白色为基调，配合柔和的自然色彩
- **暗色主题**: 以深色为基调，配合深邃的星空色彩
- **自动切换**: 根据系统主题自动切换
- **自定义**: 用户可手动选择主题

## 用户交互流程

### 主要使用场景
1. **日常项目管理**: 用户打开应用 → 查看今日项目 → 添加/编辑项目 → 设置优先级和截止时间
2. **专注工作流**: 选择项目 → 开始专注 → 进入专注模式 → 暂停/继续/停止 → 记录专注时间
3. **数据分析**: 查看统计页面 → 分析专注趋势 → 调整工作策略
4. **年度回顾**: 每年1月1日查看年度报告 → 了解全年专注情况 → 制定新年计划

### 交互特色
- **诗意反馈**: 每个操作都有诗意的文字反馈，如"时光已记录"、"专注如流水"
- **自然动画**: 使用自然元素的动画效果，如流水、飘叶、星辰
- **渐进式披露**: 复杂功能通过引导逐步展现，保持界面简洁
- **情感化设计**: 通过色彩、字体、动画传达宁静、专注的氛围
