# 静水流深 - 诗意项目管理应用

> "静水流深，专注如诗" - 将项目管理的效率工具与东方美学的诗意表达相结合

## 应用简介

"静水流深"是一款充满诗意的项目管理应用，旨在帮助用户以更加优雅和专注的方式管理时间和项目。我们相信，每一次专注都是生命中的珍贵时光，每一个项目都是成长路上的重要里程碑。

## 核心功能

### 🌊 项目管理
- **添加项目**: 支持项目名称、描述、分类、优先级、截止日期、标签
- **编辑项目**: 完整的CRUD操作
- **项目筛选**: 按分类、优先级、状态多维度筛选
- **项目搜索**: 全文搜索功能

### ⏰ 专注计时器
- **多种计时模式**: 25分钟、15分钟、45分钟、60分钟、自定义
- **专注模式**: 全屏沉浸式体验
- **状态管理**: 开始、暂停、停止、切换专注
- **时间记录**: 自动记录每次专注时长

### 📊 数据统计
- **实时统计**: 今日、本周、本月专注时间
- **趋势分析**: 专注时间变化趋势图
- **项目分布**: 各项目专注时间分布饼图
- **成就系统**: 诗意化的成就解锁

### 🎨 主题系统
- **亮色主题**: 温润的米白色背景
- **暗色主题**: 深邃的夜空背景
- **自动切换**: 跟随系统主题

### 💾 数据管理
- **本地存储**: 所有数据存储在浏览器本地
- **数据导出**: JSON格式数据备份
- **数据导入**: 支持恢复备份数据
- **归墟功能**: 一键清空所有数据

## 设计理念

### 视觉美学
- **色彩哲学**: 温润的大地色系，柔和的自然色彩
- **字体选择**: 优雅的衬线标题 + 清晰的正文字体
- **动效设计**: 流畅的过渡动画，自然的缓动曲线

### 交互体验
- **诗意反馈**: 每个操作都有诗意的文字反馈
- **自然动画**: 流水、飘叶、星辰等自然元素
- **渐进式披露**: 复杂功能通过引导逐步展现

## 技术特性

- **纯静态应用**: 无需服务器，直接在浏览器中运行
- **现代技术栈**: HTML5 + CSS3 + JavaScript ES6+
- **响应式设计**: 支持桌面、平板、移动设备
- **本地存储**: localStorage + IndexedDB
- **零依赖构建**: 无需构建工具，开箱即用

## 使用方法

### 1. 直接打开
在浏览器中打开 `index.html` 文件即可开始使用。

### 2. 本地服务器（推荐）
```bash
# 在项目目录下运行
python -m http.server 8000
# 然后访问 http://localhost:8000
```

### 3. 部署到服务器
将所有文件上传到任何支持静态文件托管的服务器即可。

## 隐私声明

- **本地存储**: 所有数据仅存储在用户本地浏览器中
- **无服务器**: 应用完全在客户端运行，不发送任何数据到服务器
- **隐私保护**: 用户的专注数据完全私密，不会被收集或分享
